## 我是谁，为什么写这个

自家有个小工厂在做出口，这几年把建站和出海工具折腾了一遍——花大几万做的官网手机端打开要 8 秒、Google 一条都不收录、hreflang 配错被判重复内容、外包跑路后代码拿不回来。找不到一份把工具和坑都摊开写清楚的清单，就自己整了这份，持续更新。觉得有用点个 Star，有补充欢迎 PR。

> **为什么现在值得做？** 这两年世界经济论坛（WEF）的"灯塔工厂"网络持续扩容，中国是入选数量最多的国家之一；制造业出海也正从"找代工、铺渠道"进入**"体系化扎根"**阶段。这意味着海外客户越来越会"先搜你公司名、再看你官网"——**SEO + GEO（生成式引擎优化）**正在变成工厂出海绕不开的基础工作，而不是加分项。

## ① 工厂官网信任要素（认证 / 资质 / 背书）

海外采购商在官网上**最先找的不是产品，是"你靠不靠谱"**。工厂实拍、第三方认证、可量化的案例数据，是把陌生流量变询盘的关键。这一栏是各大权威认证 / 信任平台的官方入口。

- [ISO（国际标准化组织）](https://www.iso.org/) — ISO 9001 质量管理体系几乎是 B2B 询盘的隐形门槛。官网能查标准内容，但**证书要找下面的第三方机构发**，别被"自我声明"忽悠。
- [CE marking（欧盟官方说明）](https://single-market-economy.ec.europa.eu/single-market/ce-marking_en) — 进欧盟市场的强制标志，官方页面写清楚哪些品类必须 CE、企业自己承担什么责任。**别只放个 CE logo 上墙，买家会查。**
- [SGS](https://www.sgs.com/) — 全球最大的检验认证机构之一，验厂报告 / 产品检测在海外买家里普遍认可。缺点：**价格偏高**，小工厂要算成本。
- [Bureau Veritas（必维）](https://www.bureauveritas.com/) — 法国老牌第三方，工业、船级、供应链审核强，欧洲客户认。
- [TÜV SÜD](https://www.tuvsud.com/) / [TÜV Rheinland](https://www.tuv.com/) — 德系认证，在机械、电子、汽配品类含金量高，做欧洲市场加分明显。
- [Intertek](https://www.intertek.com/) — 旗下 ETL 标志在**北美市场**认可度高，电子电器常用。
- [UL Solutions](https://www.ul.com/) — 北美电气安全认证，电子电器进美国基本绕不开。
- [ECHA（欧盟 REACH / SVHC 查询）](https://echa.europa.eu/) — 涉化工、材料、塑料件、纺织的工厂必看，**REACH 不合规整柜被退是真事**。
- [Trustpilot](https://www.trustpilot.com/) — 海外客户会搜你公司名看口碑，B2B 也越来越看。早点经营，比临时刷有用。
- [OpenCorporates](https://opencorporates.com/) — 全球公司主体公开数据库，适合查海外客户 / 供应商主体信息，也能提醒你把自己公司的英文主体信息整理规范。

> **【实用件】工厂官网信任要素自查表**（建站时逐条对照）

| 模块 | 及格线 | 加分项 |
|---|---|---|
| 工厂实拍 | 真实车间 / 产线照（非图库素材） | 工厂航拍视频、720° 全景、产线直播 |
| 资质认证 | ISO / CE 等**证书扫描件可点开放大** | 证书编号可在发证机构官网验真 |
| 案例数据 | "服务过哪些国家 / 行业" | 量化数字（年产能、出口国数、合作年限） |
| 团队与产能 | 员工数、厂房面积、月产能 | 关键设备清单、QC 流程图 |
| 联系信任 | 多渠道（邮箱 / WhatsApp / 微信） | 真人头像 + 姓名 + 职位，非 info@ 万能邮箱 |
| 合规背书 | 营业执照 / 工厂地址 | Google 地图可定位到真实厂区 |

## ② 产品目录与询盘系统

工厂站的转化核心就两件事：**让客户看懂产品**、**让客户方便发询盘**。表单 + 在线客服 + CRM 跟进，三件套配齐就够用。

- [WPForms](https://wpforms.com/) — WordPress 最流行的表单插件，拖拽上手快，免费版做个询盘表单够用，多步 / 条件逻辑要付费。
- [Contact Form 7](https://contactform7.com/) — 老牌免费表单，灵活但**要懂一点代码**，且默认没有反垃圾，记得配 reCAPTCHA / Turnstile。
- [Gravity Forms](https://www.gravityforms.com/) — 复杂询盘、多步骤、条件逻辑、文件上传强，纯付费，适合询盘流程复杂的工厂。
- [Tawk.to](https://www.tawk.to/) — 完全免费的在线客服聊天，预算紧可先用，缺点是功能 / 稳定性一般、移动端体验普通。
- [Crisp](https://crisp.chat/) — 在线客服 + 统一收件箱，免费版够小厂，界面现代。
- [Tidio](https://www.tidio.com/) — 客服 + 聊天机器人，能自动接首轮询盘、按规则分流，适合人手少的工厂。
- [HubSpot CRM](https://www.hubspot.com/products/crm) — 免费 CRM 管询盘跟进，邮件 / 管线 / 生态强，**高级功能贵**，但起步免费档对工厂够。
- [Zoho CRM](https://www.zoho.com/crm/) — 性价比 CRM，功能全、价格友好，缺点是界面略杂、上手有学习曲线。
- [Calendly](https://calendly.com/) — 让海外客户自助预约视频会议 / 在线验厂，**省掉时差来回约时间**的扯皮，转化体验好。
- [Cloudflare Turnstile](https://www.cloudflare.com/products/turnstile/) — 免费、隐私友好的表单防机器人方案，比传统验证码更少打扰真实采购商，询盘站强烈建议配。

## ③ 多语言本地化（hreflang / 翻译）

多语言不是"挂个 Google 翻译插件"就完事。**hreflang 配错 = Google 当你重复内容**，机翻腔 = 客户觉得你不专业。先读官方规范，再选工具。

- [Google 国际化与多语言官方文档](https://developers.google.com/search/docs/specialty/international/localized-versions) — **做多语言站之前必读**，hreflang、语言/地区版本怎么写官方说得最清楚，能避开 90% 的低级错误。
- [WPML](https://wpml.org/) — WordPress 多语言老牌插件，功能全、与主流主题 / WooCommerce 兼容好，付费，复杂电商场景首选。
- [Polylang](https://polylang.pro/) — WPML 的免费替代，基础多语言够用，复杂电商 / 字符串翻译不如 WPML 顺手。
- [TranslatePress](https://translatepress.com/) — 可视化"所见即所得"翻译，**不懂代码的人也能边看边改**，适合中小工厂站。
- [Weglot](https://www.weglot.com/) — 几行代码挂上多语言，上手最快、自动检测，缺点是**按语言数 / 流量订阅，量大偏贵**。
- [next-intl](https://next-intl.dev/) — Next.js 站做 i18n 的主流方案，开发者向，做高性能工厂站会用到。
- [DeepL](https://www.deepl.com/) — 机翻质量公认比 Google Translate 自然，**适合出初稿再让母语者校对**，别直接上线。
- [Crowdin](https://crowdin.com/) — 翻译协作平台，多人 / 持续本地化、版本管理用，团队规模大才需要。

> **【避坑】** 多语言别用纯 JS 前端翻译（客户看到的是英文，Google 抓到的还是中文）——这种"假多语言"对 SEO 等于零。要么服务端渲染、要么每个语言独立 URL + 正确 hreflang。

## ④ 展会数字化

线下展会还是工厂拿大单的核心渠道，但"展会 + 数字化"才是 2026 的玩法：展位放二维码 → 扫码进落地页 → 沉淀到 CRM / LinkedIn，把一次性见面变成长期触达。

- [广交会 Canton Fair](https://www.cantonfair.org.cn/) — 中国第一大综合外贸展，线上线下结合，工厂出海标配，线上平台也能常年挂产品。
- [Hannover Messe（汉诺威工业展）](https://www.hannovermesse.de/) — 全球最大工业展（德国），机械 / 自动化 / 工业品出海重镇。
- [Messe Frankfurt（法兰克福展览）](https://www.messefrankfurt.com/) — 旗下纺织、汽配、照明、家纺等垂直展极多，按你的品类找对口展。
- [10times](https://10times.com/) — 全球展会 / 会议目录，按行业 + 国家查"我的目标客户都去哪个展"，找新展会很好用。
- [Linktree](https://linktr.ee/) — 把官网 / 产品目录 / WhatsApp / 微信聚到一个落地页，**展位二维码、名片二维码常用**，省得印一堆链接。

## ⑤ LinkedIn / 海关数据获客

LinkedIn 是 B2B 主动获客的金矿，海关数据则适合反查真实买家和竞品客户。逻辑是：**先用数据找对公司 → 再找对人 → 内容 + 外联建立信任**。

- [LinkedIn Sales Navigator](https://business.linkedin.com/sales-solutions/sales-navigator) — 官方 B2B 找客户工具，按行业 / 职位 / 公司规模精准筛采购决策人，**贵但数据最准**。
- [Apollo.io](https://www.apollo.io/) — 找邮箱 + 外联序列一体，数据量大，有免费额度可起步，性价比高。
- [Hunter.io](https://hunter.io/) — 按公司域名找企业邮箱并验证有效性，简单直接，配合上面用。
- [Lusha](https://www.lusha.com/) — 联系人信息（邮箱 / 电话）浏览器插件，刷 LinkedIn 时顺手抓联系方式。
- [Snov.io](https://snov.io/) — 邮箱查找、验证和自动化外联一体，适合把 LinkedIn 名单变成可跟进的邮件序列。
- [ImportGenius](https://www.importgenius.com/) — 海关提单和进出口记录数据库，可反查进口商、供应商和竞品客户，适合找真实采购线索。
- [Taplio](https://taplio.com/) — LinkedIn 内容运营 / 排期 / 数据分析，适合工厂老板做"个人 IP"被动获客。

> **【避坑】** 别一上来就群发模板私信，LinkedIn 限流 + 拉黑很快。先把**公司页和个人页做成"像个真工厂"**（工厂照、产品、客户案例），再做精准外联，回复率差好几倍。

## ⑥ B2B 独立站建站工具

这是分歧最大的一栏。没有"最好"，只有"最适合你的阶段"。下面先给横向对比，再逐个点评。

| 工具 | 适合谁 | 性能 / SEO | 维护成本 | 我的一句话 |
|---|---|---|---|---|
| **WordPress** | 想快、想自己改、预算有限 | 看优化，易因插件变慢 | 中（要管安全更新） | 生态无敌，但容易堆成"插件缝合怪" |
| **Next.js** | 要长期投入、追求性能/SEO | ★ 天花板最高 | 低（上线后省心）但需开发 | 工厂站想跑赢同行，认准它 |
| **Astro** | 偏展示型内容站 | ★ 静态优先、极快 | 低 | 产品不频繁变动的工厂很合适 |
| **Webflow** | 设计自由、不想写代码 | 良 | 中（订阅费偏高） | 设计师友好，复杂功能受限 |
| **Shopify** | 要在线下单 / 小额批发 | 良 | 低 | 偏 B2C，纯询盘站用它偏重 |

- [WordPress](https://wordpress.org/) — 全球占比最高的 CMS，插件生态无敌、找人好招，缺点是**堆插件易拖慢、要持续管安全更新**。
- [Next.js](https://nextjs.org/) — React 全栈框架，性能 / SEO 上限最高，适合要长期经营、看重手机端速度的工厂站，代价是**需要开发能力**。
- [Astro](https://astro.build/) — 内容静态优先，加载极快，适合产品更新不频繁的展示型工厂站。
- [Webflow](https://webflow.com/) — 可视化建站，设计自由度高、无需写代码，订阅费偏高、复杂交互受限。
- [Framer](https://www.framer.com/) — 设计师友好、出页面快，偏营销落地页，整站功能不如上面几个。
- [Elementor](https://elementor.com/) — WordPress 可视化页面构建器，普及度高、上手快，**滥用会让页面臃肿变慢**。
- [Strapi](https://strapi.io/) — 开源 Headless CMS，常配 Next.js 做产品内容后台，开发者向。
- [Shopify](https://www.shopify.com/) — 偏 B2C 电商，工厂做样品在线下单 / 小额批发可用，纯询盘站则偏重、月费持续。

## ⑦ 海外服务器 / CDN

工厂站最常死在**"国内服务器 + 海外客户 = 打开慢到走人"**。一个海外节点 + CDN 基本解决问题。

- [Cloudflare](https://www.cloudflare.com/) — CDN / DNS / 防护一体，**免费档就能给站点全球加速 + 防攻击**，几乎是出海标配，强烈先上它。
- [Vercel](https://vercel.com/) — Next.js 官方托管，部署省心、全球边缘网络，**流量大要算账**。
- [Netlify](https://www.netlify.com/) — 静态站 / Jamstack 托管，简单易用。
- [BunnyCDN](https://bunny.net/) — 按量付费 CDN，节点多、便宜，**性价比之选**，预算敏感可选。
- [DigitalOcean](https://www.digitalocean.com/) — 海外 VPS，文档友好、社区大，适合自建可控。
- [Hetzner](https://www.hetzner.com/) — 德国主机，性价比极高（欧洲节点），**做欧洲客户首选之一**。
- [AWS](https://aws.amazon.com/) — 云全家桶，强大但复杂、**最容易超预算**，没专人别轻易上。
- [Cloudways](https://www.cloudways.com/) — 托管式云主机，帮你管好 WordPress 服务器，适合"不想碰运维"的工厂。

## ⑧ SEO / GEO 与监测工具

让 Google 找得到你、让 AI（ChatGPT / Gemini / Perplexity）愿意引用你——这就是 **SEO + GEO** 要干的事。先把免费的官方工具装齐，再考虑付费。

- [Google Search Console](https://search.google.com/search-console) — **免费、出海第一必装**，看 Google 怎么抓你站、收录情况、搜了什么词进来。
- [Google Analytics（GA4）](https://analytics.google.com/) — 免费流量分析，看访客从哪来、看了啥、在哪流失。
- [PageSpeed Insights](https://pagespeed.web.dev/) — 免费测手机 / 桌面性能分，**工厂站十有八九死在手机端慢**，先测了再说。
- [Schema.org](https://schema.org/) — 结构化数据标准，给官网打 Organization / Product 标记，**利于被 Google 摘要和 AI 引用（GEO 基础）**。
- [Ahrefs](https://ahrefs.com/) — 关键词 / 外链 / 竞品分析，专业、数据强，**贵**。
- [Semrush](https://www.semrush.com/) — 同类全能 SEO 套件，关键词库大，功能多到用不完。
- [Screaming Frog](https://www.screamingfrog.co.uk/seo-spider/) — 桌面爬虫，查死链 / 重复 title / hreflang 错误等技术 SEO 问题，免费档够小站。
- [Ubersuggest](https://neilpatel.com/ubersuggest/) — 入门级关键词工具，便宜，预算少的起步选择。
- [Bing Webmaster Tools](https://www.bing.com/webmasters/about) — 免费 SEO/GEO 工具，能看 Bing 收录、关键词、反链和站点扫描，别只盯 Google。
- [AlsoAsked](https://alsoasked.com/) — 抓 Google People Also Ask 问题，适合把采购商真实疑问整理成 FAQ、博客和产品页内容。
- [Microsoft Clarity](https://clarity.microsoft.com/) — 免费热图和会话录像，能看到海外访客在哪犹豫、乱点或放弃询盘，做转化优化很实用。

## 避坑清单与服务商怎么选

按出现频率排，这些坑最常见：

1. **手机端慢到没人等** — 海外客户大量用手机看官网。PageSpeed 手机端分低于 50 的站，等于在门口劝退客户。建站前就把性能当硬指标，别等上线了再补。
2. **假多语言坑 SEO** — 前端 JS 翻译、hreflang 配错、机翻直接上线，这三个里中一个，Google 多语言流量基本归零。
3. **只有花架子，没有询盘跟进** — 炫酷动画一堆，结果没在线客服、表单丢询盘、没 CRM 跟进。**工厂站的 KPI 是询盘数，不是好看。**
4. **代码 / 域名 / 服务器攥在外包手里** — 这是最阴的坑。合作一散，源码拿不回、域名过户被卡、想换人改不动。**签合同前一定写清：代码归你、托管在你自己的 Git 和服务器、账号你持有。**
5. **被建站平台锁死** — 某些 SaaS 建站，数据导不出、想迁站等于重做，月费还年年涨。
6. **认证只放 logo 不放证书** — 上面说过，买家会点开看、会去发证机构查，糊弄不过去。

> **关于找服务商**：模板套壳的站很便宜也很快，适合"先有个门面"的阶段；但如果你是真想靠官网拿询盘、且打算长期经营，**性能、SEO、代码归属**这三点早晚要还。我们厂自己的英文站后来是找 [模酷（ModalCube）](https://modalcube.com) 做的 Next.js，手机端 PageSpeed 跑到 90+，代码直接放在我们自己的 Git 仓库、域名服务器都在我名下——这点我特别在意，**不想再被任何人锁住**。

> **【实用件】选服务商前必问的 6 句话**
> 1. 代码和设计稿最后归谁？托管在谁的 Git / 服务器？
> 2. 手机端 PageSpeed 能保证多少分？写不写进验收标准？
> 3. 多语言怎么做？是独立 URL + hreflang，还是前端翻译？
> 4. 上线后改产品 / 加页面，是我能自己改，还是每次都得找你？
> 5. 维护期多久？过期后续费多少？出问题响应多快（有没有 SLA）？
> 6. 主体是公司还是个人？出了问题谁担责、找谁？
